# Sistema de Gestión y Reserva de Equipos Tecnológicos

## 📌 Descripción
Sistema desarrollado bajo el patrón MVC en Java para la gestión, control y reserva de equipos en laboratorios.

## 🚀 Instrucciones para ejecutar el proyecto en el IDE
1. Clonar este repositorio usando Git o descargando el ZIP.
2. Asegurarse de tener instalado **Apache NetBeans IDE** y **XAMPP/WAMP** (para MySQL).
3. Abrir NetBeans, ir a `File > Open Project` y seleccionar la carpeta descargada.
4. En XAMPP, encender el servicio MySQL, ir a phpMyAdmin y crear una base de datos llamada `bd_reservas`.
5. Ejecutar el script SQL incluido en el proyecto para crear las tablas.
6. En NetBeans, hacer clic derecho sobre la carpeta `Libraries` del proyecto > `Add JAR/Folder` y añadir el archivo `mysql-connector-j.jar`.
7. Ejecutar el archivo principal `Login.java` con Shift + F6.
